--------------------------------------------------------------


The position CSS property sets how an element is positioned in a document.
The top, right, bottom, and left properties determine the final location of positioned elements.


```
Default value     | 	static
Inherited         |   no
Animatable        |   no
JavaScript syntax |	  object.style.position="absolute"
```

# Relative

```css,playground
.target {
        position: relative;
        left: 50px; 
        top: 50px;
}
```

# Absolute

```css,playground
.target {
        position: absolute;
        left: 50px; 
        top: 50px;
}
```

# Fixed 

```css,playground
.target {
        position: fixed;
        left: 50px; 
        top: 50px;
}
```

# Sticky 

```css,playground
.target {
        position: sticky;
        top: 50px;
}
```
