--------------------------------------------------------------

shorthand for the following CSS properties:

- [border-color](./border-color)
- border-style
- border-width

# style

```css,playground,s-3,i-2
.target {
        border: solid;
}
```

# width | style

```css,playground,s-3,i-2
.target {
        border: 2px dotted;
}
```

# style | color

```css,playground,s-3,i-2
.target {
        border: outset #f33;
}
```

# width | style | color

```css,playground,s-3,i-2
.target {
        border: medium dashed green;
}
```
